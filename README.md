# PIBOOK – Website bán sách online (Đồ án tốt nghiệp)

## Giới thiệu
PIBOOK là website thương mại điện tử bán sách trực tuyến, được xây dựng hoàn toàn bằng React + Next.js 14 (App Router) + TypeScript + Node.js + MySQL.

## Tính năng chính
- Tìm kiếm & lọc sách theo thể loại, tác giả, giá
- Giỏ hàng + thanh toán COD / chuyển khoản / VNPay
- Mã giảm giá + miễn phí ship đơn từ 500.000đ
- Trang quản trị đơn hàng (Chờ xác nhận → Đang xử lý → Đang giao → Hoàn thành / Hủy)
- Đăng nhập người dùng, lưu lịch sử mua hàng
- Responsive 100% (mobile + desktop)

## Công nghệ sử dụng
**Frontend**
- Next.js 14 (App Router)
- React + TypeScript
- Bootstrap 5 + React-Bootstrap
- react-hot-toast, react-icons

**Backend**
- Node.js + Express
- MySQL
- JWT authentication

**Thanh toán**
- VNPay (QR + chuyển hướng)
- COD & chuyển khoản thủ công
